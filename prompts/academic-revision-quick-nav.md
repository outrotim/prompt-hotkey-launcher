> **本文件是入口层,不是主干。** 所有具体逻辑、输出模板、任务边界都在 `academic-revision-system.md`。
> 任一条入口严格封顶三要素:**用途 / 对应步骤 / 触发语**。新增条目若超出三要素即视为向模板层回流,必须回退。

# 写作前快启

## 主线定位
- 用途:分析已完成、尚未正式写正文时,确定最稳妥主线
- 对应:`academic-revision-system / A1`
- 触发语:`现在执行第A1步。`

## 结果分层
- 用途:把现有结果按论文叙事价值分成主核/延伸/支撑/补充
- 对应:`academic-revision-system / A3`
- 触发语:`现在执行第A3步。`

## 论文蓝图
- 用途:基于主线锚点和结果层级,生成写作前蓝图
- 对应:`academic-revision-system / A5`
- 触发语:`现在执行第A5步。`

## 写作清单
- 用途:生成可直接执行的写作施工清单
- 对应:`academic-revision-system / A6`
- 触发语:`现在执行第A6步。`

## 阶段交接
- 用途:正文初稿接近完成时,决定能否切换到修稿阶段,或回退到 A2–A6
- 对应:`academic-revision-system / A7`
- 触发语:`现在执行第A7步。`

# 修稿投稿快启

## 总诊断
- 用途:已成稿后,把整篇稿件收敛成一份高质量修改总纲
- 对应:`academic-revision-system / B0`
- 触发语:`现在执行第B0步。`

## 引言重建
- 用途:基于修改总纲和主线锚点,重建 Introduction 的逻辑与段落顺序
- 对应:`academic-revision-system / B1`
- 触发语:`现在执行第B1步。`

## 结果重排
- 用途:按主线强度重新排列 Results,不先润色语言
- 对应:`academic-revision-system / B3`
- 触发语:`现在执行第B3步。`

## 讨论重建
- 用途:重建 Discussion 的论证链与 limitation 布局
- 对应:`academic-revision-system / B4`
- 触发语:`现在执行第B4步。`

## 因果措辞校准
- 用途:逐句校准因果措辞、机制表述、弱结果写强、跨版块强度漂移
- 对应:`academic-revision-system / B6`
- 触发语:`现在执行第B6步。`

## 投稿终检
- 用途:首投前按期刊编辑和审稿人标准做最后终检
- 对应:`academic-revision-system / B11`
- 触发语:`现在执行第B11步。`

## 审稿意见分流
- 用途:收到 reviewer comments 后,先分流再设计动作,不直接写回复信
- 对应:`academic-revision-system / B12`
- 触发语:`现在执行第B12步。`

## 修回前终检
- 用途:修回稿提交前,检查回复信—正文一致性与引用定位是否仍有效
- 对应:`academic-revision-system / B13`
- 触发语:`现在执行第B13步。`

# 常用工具(转发入口)

## 单段改写
- 用途:只处理某一段文字,不扩展到全文
- 对应:`academic-revision-system / §03 / 单段改写模式`
- 触发语:请参照 system §03 单段改写模式执行。

## 摘要重写
- 用途:只处理摘要,不扩展到正文
- 对应:`academic-revision-system / §03 / 摘要重写模式`
- 触发语:请参照 system §03 摘要重写模式执行。

## 标题生成
- 用途:生成候选标题,按稳健/结果/期刊适配分类
- 对应:`academic-revision-system / §03 / 标题生成器`
- 触发语:请参照 system §03 标题生成器执行。

## Cover Letter
- 用途:生成投稿 Cover Letter
- 对应:`academic-revision-system / §03 / Cover Letter 生成`
- 触发语:请参照 system §03 Cover Letter 生成执行。

## 审稿回复
- 用途:起草审稿意见回复信。**若尚未做审稿意见分流,请先执行第B12步。**
- 对应:`academic-revision-system / §03 / 审稿人回复信`
- 触发语:请参照 system §03 审稿人回复信执行。

## AI 痕迹整理
- 用途:清理明显 AI 写作痕迹,但不改变事实、立场和结论边界
- 对应:`academic-revision-system / §03 / AI 痕迹整理`
- 触发语:请参照 system §03 AI 痕迹整理执行。
